Emotional Wellness App

Project ChatBot under guidance Prof.J Dinesh Babu

Overview

This is an emotional wellness chatbot that has an automatic generated trained conversation; Web and PWA application. The chatbot is trained by the dataset given below ( with reference from Dr. Gangadhar Garipelli ) and can be downloaded.

Hightlights

  1. Done using Google Dialogflow API
  2. The API is linked in Nodejs application where node acts as an interface between client and DialogFlow
  3. Follow Up intends are handled in DialogFlow.
  4. User-Friendly and Responsive
  5. Audio to Text and Text to Audio is also available.
  6. PWA access is available.
